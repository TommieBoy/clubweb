Repeater Network

The KARC repeater network provides wide-area coverage across the Kamloops region and surrounding communities. All repeaters are linked via amateur radio broadband IP network for regional, provincial, national, and international connectivity.

Coverage Area

The KARC repeater network provides broad coverage to:

  • Kamloops
  • Logan Lake
  • Merritt
  • Ashcroft
  • Trans-Canada Highway #1 (south to Lytton)
  • Yellowhead Highway #5 (north to Barriere)

Network Status & Linking

All repeaters are linked via amateur radio broadband IP network, enabling:

  • Regional communication between mountain-top sites
  • Provincial and national network access
  • International connectivity via WIRES-X and AllStar

Note: Occasional maintenance windows may affect linked network availability. Check karc@karc.ca for scheduled maintenance announcements.

For Visitors

The KARC repeater system is open for use by all visiting licensed amateur operators. All repeaters are accessible with standard FM rigs or digital-capable radios. Digital modes (Fusion/C4FM) are supported on most repeaters.
