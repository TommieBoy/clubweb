# Navigation Plan

## Goals

- Help visitors find stories in 1-2 clicks.
- Keep top navigation simple and task-focused.
- Reduce admin effort by grouping low-use links.

## Proposed Main Navigation

1. Stories
2. Calendar
3. Repeaters
4. About
5. Join
6. Contact
7. Resources

## Resources Dropdown

- Club Constitution
- Club Personnel
- Meeting Minutes
- Forms
- Archive

## Navigation Rules

- Keep top-level items to 7 max.
- Use action words when possible (`Join`, `Contact`).
- Keep labels short (1-2 words).
- Any page not used weekly moves into `Resources`.

## Story Discovery Links

- Add `Latest Stories` and `Story Topics` links under `Stories`.
- Add a `View all stories` button on homepage hero.
- Keep breadcrumbs enabled on story pages.

## URL Pattern

- Stories: `/stories/<slug>`
- Events: `/events/<slug>`
- Static pages: `/about`, `/join`, `/contact`
