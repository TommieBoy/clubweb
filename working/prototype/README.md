# Prototype Home Page

Files in this folder are a standalone prototype focused on:

- Faster story discovery for visitors
- Easier admin updates for stories, events, and navigation

## Files

- `index.html`: prototype layout
- `style.css`: visual style and responsive behavior

## Notes

- This is a front-end mockup for planning and review.
- It does not replace Drupal templates directly.
- Use `working/proposal/drupal-action-list.md` for CMS implementation steps.
